## usage

```bash
npm i
npm run dev
```

## use skill
```
Frontend: React.js,  Next.js
UI Framework: MaterialUI
Backend: Express, Apollo Server
Lauguage: TypeScript
```
## schema

```gql
  type User {
    id: ID!
    workId: String!
    password: String!
    name: String!
    manager: ID
    position: String!
  }

  type Task {
    id: ID!
    title: String!
    createdBy: User!
    viewers: [ID]!
    editors: [ID]!
    status: String!
    content: String!
  }
```

## json data storage
users: /apollo/data/users

tasks: /apollo/data/tasks
